# Three.js Basics

This project demonstrates the fundamentals of Three.js by creating a simple animated 3D cube.

## Features

- Create a 3D Scene
- Perspective Camera
- WebGL Renderer
- Box Geometry
- MeshStandardMaterial
- Ambient, Directional, and Point Lighting
- Cube Rotation Animation
- Responsive Window Resize

## Technologies Used

- HTML5
- JavaScript (ES6)
- Three.js (r128)

## Concepts Covered

- Scene
- Camera
- Renderer
- Geometry
- Material
- Mesh
- Lights
- Animation Loop
- Window Resize Handling
